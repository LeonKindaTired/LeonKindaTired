### What I'm building

At Internstellar, we rethink how talent is discovered and validated.

Traditional hiring is broken:

* CVs do not reflect real ability
* Technical interviews are inconsistent
* Companies waste time filtering instead of evaluating

We are building a system where:

* Candidates prove skills through real tasks
* Companies see actual performance before hiring
* Hiring decisions are based on signal, not assumptions
